# Export Customization - Visual Guide

## Overview
This guide provides visual representations of the export customization feature, showing how users can personalize their weekly agenda exports.

```

└─ Settings Icon (right) ← Click here
    │
    └─ Settings Sheet Opens
        │
        ├─ ⭐ Activity Templates
        ├─ 🔗 Calendar Subscriptions
```

```
┌──────────────────────────────────┐
```

```
- Text Color: #003366 (Corporate blue)
- Background: Company logo (low opacity)
- Accent Color: #003366
- Border Radius: 4px

Professional export matching company brand
```

```
Settings:
- Header Font: Georgia

- Text Color: #2d3436

- Background Opacity: 70%
- Accent Color: #d63031 (Red)

Warm, seasonal autumn-themed export

```

## Tips

✅ **Test your design** in the export preview before exporting
✅ **Use subtle backgrounds** for better text readability
✅ **Adjust opacity** if background image is too strong
✅ **Save time** - settings persist automatically
✅ **Reset anytime** with the "Reset to Defaults" button

## Summary

The export customization system provides a simple, visual way to personalize weekly agenda exports with:

- 🎨 Custom fonts and colors
- 🖼️ Background images
- ⚙️ Styling controls
- 👁️ Real-time preview
- 💾 Persistent settings
- 🔄 Easy reset

All accessible through an intuitive settings interface!
